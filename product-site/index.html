What it does

  • Runs faceless. No Dock icon and no menu-bar item — only the island.
  • Real-time status. Agents post status, task, and progress over a local HTTP server.
  • Blocks for approval. A synchronous request holds an agent until you approve or reject.
  • Hooks into Claude Code. Shell hooks mirror Claude Code's status into the island.
  • Python drop-in. eidos.py wraps the OpenAI Agents SDK with IslandHook and approval_tool.
  • Dependency-free. SwiftUI + AppKit + Network.framework. Builds with the Command Line Tools.

Protocol — localhost:7799

POST /event    status update; fire-and-forget
POST /approve  blocks until you approve / reject
GET  /status   what's running now

Run it

# macOS 14+ · no full Xcode required
git clone https://github.com/yikolim/eidos-agent-island
cd eidos-agent-island
./build.sh --run
